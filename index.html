Simple BlockChain Simulation

This project is a basic blockchain simulation implemented in Java, demonstrating fundamental blockchain concepts such as block creation, secure hashing, proof-of-work, and tamper detection. Each block is linked securely using SHA-256 hashing, ensuring data integrity. The simulation supports mining with an adjustable proof-of-work difficulty, allowing users to understand the computational effort required for block validation. Additionally, the system verifies the integrity of the blockchain and detects any tampering when data is modified, making it a useful educational tool for exploring blockchain security and functionality.

Student Management System

This Java console application efficiently manages student records while demonstrating object-oriented programming (OOP) principles. It organizes student data using classes like `Student` and `Subject`, allowing users to add, display, search, and delete student records. The system automatically calculates the average marks for each student based on their subject scores. With a structured approach, the `StudentManager` class handles all operations, while a menu-driven interface in the `Main` class ensures smooth user interaction. This project provides a practical example of OOP concepts in Java, making it a useful tool for academic record management.

EchoBot - AI Chatbot

EchoBot is a dynamic Android chatbot application designed to provide an interactive conversational experience. Built using Java in Android Studio, it integrates a FastAPI backend with LangChain and a local LLM powered by Ollama. The frontend features a sleek chat interface using `RecyclerView`, `EditText`, and `Send Button`, ensuring seamless message flow. Retrofit and OkHttp handle API communication, while the `ChatAdapter` efficiently manages chat bubbles. With an auto-scrolling chat view and a structured `BotResponse` model, EchoBot delivers smooth and intelligent interactions. The backend is hosted on free cloud services like Render, making it a cost-effective AI chatbot solution.
